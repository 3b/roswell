NAME

Roswell - Common Lisp environment setup Utility

SYNOPSIS

ros[options][[command [args...]]|[[--]script [args...]]]

DESCRIPTION

Roswell is a command line tool to install and manage Common Lisp implementations damn easily.

OPTIONS

--version:Prints the Roswell version

-m,--image:Specify image file name

ROS COMMANDS

install:Download archive and build it for roswell environment.

setup:Initial setup

config:Get and Set options for Roswell

version:Show the roswell version information

help:Show Command help

run:Run repl
